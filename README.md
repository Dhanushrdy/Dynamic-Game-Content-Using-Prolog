# Dynamic-Game-Content-Using-Prolog
Download and Install SWI-Prolog
SWI-Prolog is the industry-standard environment for logic programming. It provides the compiler and the interactive shell required to run .pl files.

Official Download Page: SWI-Prolog Downloads

Direct Links:

Windows (64-bit): [Download .exe](https://www.swi-prolog.org/download/stable/bin/swipl-10.0.2-1.x64.exe.envelope)

macOS (Intel/Apple Silicon): [Download .dmg](https://www.swi-prolog.org/download/stable/bin/swipl-10.0.2-1.fat.dmg)

Linux: Use the terminal: sudo apt install swi-prolog



Running the Source File:
Launch the SWI-Prolog application.

Go to File > Consult...

Navigate to your Git directory and select dragon_slayer.pl.

If the bottom of the window says true., your logic is loaded correctly.



Standard Commands:

Movement: w., s., a., d.

Interaction/Grab: f.

Combat/Attack: e.

Check Status: look. or inventory.

Exit: Type halt. to close the Prolog interpreter.
